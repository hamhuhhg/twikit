# برنامج إدارة حسابات تويتر

هذا البرنامج هو أداة لإدارة حسابات تويتر (X) وتنفيذ التفاعلات المختلفة بشكل آلي.

## الميزات

- **إدارة الحسابات:** إضافة وتعديل وحذف حسابات تويتر.
- **النشر الآلي:** نشر التغريدات من الحسابات الرئيسية.
- **التفاعلات الآلية:** تنفيذ تفاعلات مثل الإعجاب وإعادة التغريد والتعليق من الحسابات الفرعية.
- **التحليلات:** عرض تقارير حول أداء الحسابات.
- **نسخ متعددة:** نسخ مختلفة من البرنامج بصلاحيات مختلفة.

## المتطلبات

- Python 3.8+
- PyQt5
- twikit

## التثبيت

1.  استنساخ المستودع:
    ```bash
    git clone https://github.com/d60/twikit.git
    cd twikit
    ```
2.  تثبيت المتطلبات:
    ```bash
    pip install -r requirements.txt
    pip install PyQt5
    ```

## الاستخدام

1.  **إنشاء قاعدة البيانات:**
    ```bash
    python database.py
    ```
2.  **تشغيل النسخة الرئيسية:**
    ```bash
    python main_window.py
    ```
3.  **تشغيل نسخة العاملين:**
    ```bash
    python worker_main_window.py
    ```
4.  **تشغيل نسخة التحليلات:**
    ```bash
    python analytics_only_window.py
    ```

## النسخ

- **النسخة الرئيسية (`main_window.py`):** تحتوي على جميع الميزات، بما في ذلك إدارة الحسابات والنشر والتفاعلات والتحليلات.
- **نسخة العاملين (`worker_main_window.py`):** تحتوي على جميع الميزات باستثناء واجهة التحليلات.
- **نسخة التحليلات (`analytics_only_window.py`):** تحتوي فقط على واجهة التحليلات والمشاهدات.
